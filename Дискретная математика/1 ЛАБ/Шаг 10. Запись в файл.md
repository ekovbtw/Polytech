Нужно записать в файл итоговую матрицу. 
для этого поменяем все printf на fprintf
пример: 
`fprintf(out, "%d\n", summ);`
И не забыть объявить файл для записи: 
`FILE* out = fopen("input_5.out", "w");`

`void add_to_file()`
`{`
    `FILE* out = fopen("input_5.out", "w");`
    `if (out == NULL)`
    `{`
        `printf("Error: cannot create output file\n");`
        `return;`
    `}`
    `// 1. сумма`
    `fprintf(out, "%d\n", summ);`
    `// 2. матрица остова`
    `for (int i = 0; i < kol_vershin; i++)`
    `{`
        `for (int j = 0; j < kol_vershin; j++)`
        `{`
            `fprintf(out, "%d", end_matrix[i][j]);`
            `if (j != kol_vershin - 1)`
            `{`
                `fprintf(out, ", ");`
            `}`
        `}`
        `fprintf(out, "\n");`
    `}`
    `// 3. рёбра`
    `for (int i = 0; i < kol_vershin - 1; i++)`
    `{`
        `fprintf(out, "(%d, %d)", E[i][0] + 1, E[i][1] + 1);`
        `if (i != kol_vershin - 2)`
        `{`
            `fprintf(out, ", ");`
        `}`
    `}`
    `fprintf(out, "\n");`
    `fclose(out);`
`}`
И просто в конце в main объявим ее.
`add_to_file();`
