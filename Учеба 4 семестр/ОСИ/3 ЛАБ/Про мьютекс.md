Из методички:

> Для захвата мутекса необходимо просто дождаться перехода в сигнальное состояние с помощью функции WaitForSingleObject(). Если нить захватила мутекс, то его освобождение осуществляется с помощью функции ReleaseMutex().

Итого:

- Захватить мутекс — WaitForSingleObject(mutex, INFINITE)
- Отпустить мутекс — ReleaseMutex(mutex)
