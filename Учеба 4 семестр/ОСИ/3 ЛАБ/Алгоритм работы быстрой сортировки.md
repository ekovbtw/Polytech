## QuickSort — многопоточная сортировка (Windows, Mutex + Event)

### 1. Алгоритм QuickSort — объяснение по-человечески

QuickSort работает по принципу «разделяй и властвуй»:

1. Выбираешь один элемент — **опорный (pivot)**
2. Всё, что меньше него — перемещаешь влево, всё что больше — вправо
3. Pivot теперь стоит на своём окончательном месте
4. Рекурсивно делаешь то же самое для левой и правой части

**Пример на `массиве [8, 3, 1, 5, 2]`:**

```
Шаг 1: pivot = 5
  [8, 3, 1, 5, 2]
        ↓
  меньше 5 → [3, 1, 2]  |  [5]  |  больше 5 → [8]

Шаг 2: сортируем [3, 1, 2], pivot = 2
  меньше 2 → [1]  |  [2]  |  больше 2 → [3]

Шаг 3: [1] и [3] — по одному элементу, готово

Итог: [1, 2, 3, 5, 8] ✓
```
