### Пункт 2. Глобальные переменные и объекты синхронизации

Тебе нужно объявить всё, что будет общим для всех потоков. Это делается **глобально** (до), потому что все потоки должны иметь к этому доступ.`main`

**Что нужно объявить:**

- `int* global_array` — сам массив (потоки будут его сортировать)
- `int N` — размер массива
- `int global_count_threads` — количество потоков
- `HANDLE g_mutex` — мутекс для защиты очереди задач
- `HANDLE g_event` — событие «появилась новая задача» (автосброс)
- `HANDLE g_done_event` — событие «вся работа завершена» (ручной сброс)
- Структура — описывает одну задачу (левый и правый индекс куска массива, указатель на следующий элемент очереди)`Task`
- `Task* g_queue_head` — голова очереди задач
- `int g_queue_cnt` — количество задач в очереди
- `int g_active` — количество потоков, которые сейчас активно работают
  
Я немного добавил своего стиля в этот пункт, поэтому вышло так: 

int global_count_threads = 0; // количество потоков
int N = 0; // количество элементов массива 
int* global_array = NULL; // массив где хранятся числа для быстрой сортировки

typedef struct
{
    int left; // левая граница подмассива
    int right; // правая граница подмассива
} Task;

HANDLE mutex; // мьютекс для синхронизации доступа к глобальному массиву
HANDLE event; // событие для уведомления потоков о том, что массив отсортирован
HANDLE done_event; // событие для уведомления о завершении работы потоков
volatile int active_threads = 0; // количество активных потоков

// подробнее о volatile - [[Volatile]]

Task* task_array = NULL; // очередь задач для сортировки
volatile int head = 0; // индекс головы очереди задач
volatile int tail = 0; // индекс хвоста очереди задач

void update_memory(int count)
{
    Task* buff = (Task*)realloc(task_array, count * sizeof(Task));
    if (buff == NULL)
    {
        // обработка ошибки — старый task_array остался целым
        printf("Не удалось перевыделить память\n");
        return;
    }
    task_array = buff;
}
