
Установить инструменты:
- `yasm`
- `gcc`
- `ld`
- `qemu-system-i386`

Без них вы не сможете:

- собрать загрузчик,
- собрать ядро,
- запустить всё в эмуляторе.

## Команды

`sudo apt update`  
`sudo apt install yasm`
`sudo apt install qemu-system-i386
`sudo apt install g++`
`sudo apt install gcc-multilib g++-multilib`
