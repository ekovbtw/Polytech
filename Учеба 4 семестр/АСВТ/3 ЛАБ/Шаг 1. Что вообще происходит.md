Это **таймер обратного отсчёта (ММ.СС)**.

Он:

1. отображает время на **4 семисегментных индикаторах**
2. использует **динамическую индикацию**
3. использует **2 таймера**
4. использует **внешние прерывания кнопок**

Кнопки:

| Кнопка         | Назначение      |
| -------------- | --------------- |
| **PD2 (INT0)** | старт / пауза   |
| **PD3 (INT1)** | режим настройки |
| **PD0**        | увеличить       |
| **PD1**        | уменьшить       |

Когда таймер достигает **00.00**  
→ индикатор **мигает 2 Гц**
