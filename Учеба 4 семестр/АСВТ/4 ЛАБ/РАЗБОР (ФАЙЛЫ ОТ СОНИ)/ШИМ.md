## Широтно-импульсная модуляция 

Микроконтроллер умеет выдавать только 0В или 5В. Получить промежуточное напряжение, например 2.5В, напрямую он не может. ШИМ решает проблему через **быстрое переключение**.  То есть если очень быстро включать и выключать сигнал в нужной пропорции - на выходе получается **эффект среднего напряжения.**

### Скважность 
 Это процент времени, когда сигнал находится в состоянии HIGH (5В), за один период. 
 
 90% скважность:
‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾|__| 
←      90%        →←10%→
Среднее напряжение = 5В × 0.9 = 4.5В → светодиод яркий

50% скважность:
‾‾‾‾‾‾‾‾‾‾|__________|
←   50%   →←   50%   →
Среднее напряжение = 5В × 0.5 = 2.5В → светодиод средний

10% скважность:
‾‾|__________________|
←10%→←    90%       →
Среднее напряжение = 5В × 0.1 = 0.5В → светодиод тусклый

### Как ШИМ реализуется через таймер

Внутри микроконтроллера таймер непрерывно считает от 0 до TOP. Его текущее значение сравнивается с числом в регистре OCR:

OCR=64          TOP=255
          ↓               ↓
0 ────────|───────────────|── 0 ────...

Счётчик < OCR → выход PIN = HIGH (5В)
Счётчик > OCR → выход PIN = LOW  (0В)

Меняя значение OCR — меняешь скважность:
OCR=0   → скважность 0%   → всегда LOW  → светодиод не горит
OCR=128 → скважность 50%  → половину HIGH → средняя яркость
OCR=255 → скважность 100% → всегда HIGH → максимальная яркость

### Режим Fast-PWM

В лабораторной используется режим **Fast-PWM** — самый простой и быстрый режим ШИМ.

Принцип работы:

```
Счётчик считает ВВЕРХ от 0 до TOP:

  TOP ───────────────╮
                     │
  OCR ──────╮        │
            │        │
  0 ────────╯────────╯────...
  
  Выход:  ‾‾‾‾‾‾|___|‾‾‾‾‾‾|___|
               HIGH LOW   HIGH LOW
  
  Когда счётчик = 0   → выход = HIGH
  Когда счётчик = OCR → выход = LOW
  Когда счётчик = TOP → сброс в 0, повтор
```


### Регистры для работы с ШИМ

В варианте 5а используются **два таймера** — T1 и T2, управляющие двумя светодиодами.

#### Таймер T2 → управляет ножкой OC2 (PD7)

**TCCR2** — регистр управления таймером T2:

```
TCCR2: [ FOC2 | WGM20 | COM21 | COM20 | WGM21 | CS22 | CS21 | CS20 ]
         бит7   бит6    бит5    бит4    бит3    бит2   бит1   бит0
```

**WGM21, WGM20** — выбор режима работы таймера:

```
WGM21=1, WGM20=1 → режим Fast-PWM
```

**COM21, COM20** — поведение вывода OC2 при совпадении:

```
COM21=1, COM20=0 → при совпадении OC2=LOW, при сбросе OC2=HIGH
                   (неинвертированный ШИМ — чем больше OCR2, тем ярче)
```

**CS22, CS21, CS20** — делитель частоты таймера:

```
CS22=1, CS21=0, CS20=0 → делитель /64
```

Пример настройки:

asm

```asm
LDI R16, (1<<WGM20)|(1<<WGM21)|(1<<COM21)|(1<<CS22)
OUT TCCR2, R16
```

**OCR2** — регистр скважности для T2:

asm

```asm
LDI R16, 128      ; скважность ~50%
OUT OCR2, R16     ; записываем в регистр сравнения
```

---

#### Таймер T1 → управляет ножкой OC1B (PD4)

Таймер T1 — 16-битный, но для ШИМ используем его в 8-битном режиме. Настраивается через **два регистра**:

**TCCR1A** — первый регистр управления:

```
TCCR1A: [ COM1A1 | COM1A0 | COM1B1 | COM1B0 | FOC1A | FOC1B | WGM11 | WGM10 ]
```

```
WGM10=1           → Fast-PWM 8-бит (TOP=255)
COM1B1=1, COM1B0=0 → неинвертированный ШИМ на OC1B
```

**TCCR1B** — второй регистр управления:

```
TCCR1B: [ ICNC1 | ICES1 | - | WGM13 | WGM12 | CS12 | CS11 | CS10 ]
```

```
WGM12=1           → часть настройки Fast-PWM
CS11=1, CS10=1    → делитель /64
```

Пример настройки:

asm

```asm
LDI R16, (1<<WGM10)|(1<<COM1B1)
OUT TCCR1A, R16

LDI R16, (1<<WGM12)|(1<<CS11)|(1<<CS10)
OUT TCCR1B, R16
```

**OCR1BL** — регистр скважности для T1 канал B (8 бит):

asm

```asm
LDI R16, 200      ; скважность ~78%
OUT OCR1BL, R16
```
