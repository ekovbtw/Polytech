На каждом из регистров PORTA, PORTB и PORTC отображается «бегущий огонь» из набора горящих светодиодов. Для каждого порта существует базовое значение (набор одновременно горящих светодиодов), которое с определённой частотой (для каждого регистра своей) циклически сдвигается на определённый шаг (для каждого регистра свой). Параметрами гирлянды являются:  базовое значение (b) – положительное число, которое циклически сдвигается и отображается на соответствующем регистре;  величина шага (h), на которую циклически сдвигается базовое значение каждый такт гирлянды;  частота смены состояния (p);  начальное смещение (d), определяющее с каким значением смещения начинает работать гирлянда при запуске программы. Все параметры задаются для регистров PORTA, PORTB и PORTC в виде «ba.», «bb.», «bc.» и т.д. Примеры отображения настроек на семисегментном индикаторе приведены на рисунке 2.

### Параметры гирлянды.

То есть параметры гирлянды: 
**b = base** - базовое значение, какие светодиоды горят одновременно, 
**h = hop/step** - шаг сдвига, на сколько позиций сдвигается один такт, 
**p = period** - частота, как часто происходит сдвиг, 
**d = delay** - начальное смещение, с какой позиции стартует гирлянда. 
Например: pb=3 означает частота порты В = 3 раза за период.
### Как бежит огонь на каждом порту

#### PORTA
**(ba = 3 = 0b00000011, ha = 2, pa = 4, da = 0)

ba=3 → горят два соседних светодиода PA0 и PA1
ha=2 → каждый такт сдвигается на 2 позиции влево
pa=4 → сдвиг происходит каждые 4 тика таймера
da=0 → стартует с нулевого смещения

Движение:
00000011 → 00001100 → 00110000 → 11000000 → 00000011 ...

#### PORTB
**(bb=0x0F=0b00001111, hb=1, pb=настраивается, db=настраивается):

bb=0x0F → горят четыре соседних светодиода PB0-PB3
hb=1      → каждый такт сдвигается на 1 позицию влево
pb          → настраивается потенциометром 1..8
db          → настраивается потенциометром 0..3

Движение при pb=1, db=0:
00001111 → 00011110 → 00111100 → 01111000 → 11110000 → 00001111 ...

#### PORTC
**(bc=6=0b00000110, hc=1, pc=6, dc=1):

bc=6  → горят светодиоды PC1 и PC2
hc=1  → сдвиг на 1 позицию
pc=6  → каждые 6 тиков
dc=1  → стартует уже сдвинутым на 1 шаг

Стартовое положение (после 1 сдвига):
00000110 → сдвиг на 1 → 00001100 (это начальная позиция)

### Два режима работы

**Режим 0 — гирлянда работает:**

Все три порта независимо мигают своими огнями
Кнопки PD2 и PD3 игнорируются (кроме переключения режима)
Потенциометр игнорируется

**Режим 1 — настройка параметров:**

Гирлянда останавливается
На семисегментном индикаторе отображается имя и значение параметра
Потенциометр меняет значение текущего параметра

### Кнопки

**PD2 (INT0)** — переключение между режимами:

Нажал → гирлянда → настройка
Нажал → настройка → гирлянда (с новыми параметрами)

**PD3 (INT1)** — переключение между параметрами (только в режиме настройки):

Нажал → pb (частота)
Нажал → db (смещение)
Нажал → bb (базовое значение)
Нажал → pb (снова по кругу)

### Что показывает индикатор в режиме настройки

Четыре семисегментных индикатора показывают:

```
Индикатор: [1]  [2]  [3]  [4]
           имя  порт  пуст значение

pb → [p]  [b.] [ ]  [значение 1-8]
db → [d]  [b.] [ ]  [значение 0-3]
bb → [b]  [b.] [ ]  [значение 0-F hex]
```

### Светодиоды ШИМ PD7 и PD4

Показывают **где находится текущий параметр** в своём диапазоне:

```
pb=1 (минимум): PD7 = максимум яркости, PD4 = 0
pb=4 (середина): PD7 = средняя яркость, PD4 = средняя яркость
pb=8 (максимум): PD7 = 0, PD4 = максимум яркости
```

Визуально это как **индикатор положения** — два светодиода всегда дополняют друг друга до максимума.
