### 1. Частота процессора 
```
#define F_CPU 8000000UL
```
Это говорит компилятору что процессор работает на **8 МГц**. Нужно для правильной работы задержек. Должно быть **самой первой строкой**.

### 2. Библиотеки 
```
#include <avr/io.h>
#include <util/delay.h>
#include <avr/interrupt.h>
```
**avr/io.h** — даёт доступ к регистрам микроконтроллера (DDRA, PORTA, TCCR0 и т.д.). Без неё компилятор не знает что такое PORTA или OCR2.

**util/delay.h** — функции задержки (_delay_ms). Нужна F_CPU чтобы правильно считать время.

**avr/interrupt.h** — даёт возможность писать обработчики прерываний ISR(...) и использовать sei().

### 3. Порядок написания кода 

1. define F_CPU, include - библиотеки
2. Глобальные переменные
3. Таблица символов и Функции (crcl, update_garland, init_garlands, timer_init, ADC_init, take_adc)
4. Обработчики прерываний ISR(...)
5. main()
