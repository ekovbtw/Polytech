#### Суть логики №2

В логике №2 каждый порт живёт сам по себе. У порта A — свои параметры, у порта B — свои, у порта C — свои. Они не зависят друг от друга.

На каждом порту — «бегущий огонь». Это значит: горит какой-то набор светодиодов, и через какое-то время они все вместе сдвигаются.

---

#### Четыре параметра каждого порта

Каждый порт описывается четырьмя вещами:

- b — базовое значение. Это маска — какие светодиоды горят изначально. Например, bb = 0x0F — это 0b00001111 — горят биты 0, 1, 2, 3 — то есть первые четыре светодиода порта B.
- h — шаг сдвига. На сколько бит сдвигается маска за один такт гирлянды. Например, hb = 1 — каждый раз сдвигаем на 1 бит влево.
- p — частота. Через сколько тактов гирлянды происходит один сдвиг. Например, pb = 1 — сдвиг каждый такт. pb = 4 — сдвиг каждые 4 такта.
- d — начальное смещение. На сколько бит сдвинута маска в самом начале, при запуске программы.

---

#### Пример на порту B из кода

Смотри на переменные в коде:

```
bb = 0x0F   → горят биты 0,1,2,3
hb = 1      → сдвиг на 1 бит
pb = 1      → сдвиг каждый такт (настраивается потенциометром, диапазон 1..8)
db = 0      → начинаем без смещения
```

Как это выглядит во времени:

```
Такт 0:  0b00001111  → горят светодиоды 0,1,2,3
Такт 1:  0b00011110  → горят светодиоды 1,2,3,4
Такт 2:  0b00111100  → горят светодиоды 2,3,4,5
Такт 3:  0b01111000  → горят светодиоды 3,4,5,6
Такт 4:  0b11110000  → горят светодиоды 4,5,6,7
Такт 5:  0b11100001  → огонь "обернулся" — это циклический сдвиг
Такт 6:  0b11000011
...и так по кругу
```

Огонь бежит влево и когда доходит до края — появляется с другой стороны. Это и есть циклический сдвиг.

---

#### Твой вариант 5а

По таблице из методички, вариант 5, параметры порта B:

```
bb = 0x0F  (неизменяемый, в коде написано bb=f, f = 0x0F)
hb = 1     (неизменяемый)
db = 0     (неизменяемый)
pb         (НАСТРАИВАЕМЫЙ, диапазон 1..8, раз в 2 секунды)
```

Именно pb ты и меняешь потенциометром. Это скорость движения огня на порту B.

---

#### Что значит "раз в 2 секунды"

В методичке написано: pb — частота, раз в 2 секунды. Это значит pb — это количество сдвигов за 2 секунды. Если pb = 1 — один сдвиг за 2 секунды (медленно). Если pb = 8 — восемь сдвигов за 2 секунды (быстро).
