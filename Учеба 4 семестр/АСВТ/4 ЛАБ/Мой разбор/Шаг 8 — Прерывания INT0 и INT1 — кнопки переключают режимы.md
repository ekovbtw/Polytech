#### Что такое внешнее прерывание

Внешнее прерывание — это когда сам микроконтроллер следит за ножкой. Как только на ней что-то изменилось — он бросает всё что делает и вызывает нужную функцию.

В коде две кнопки:

- PD2 — прерывание INT0 — переключение между режимом гирлянды и режимом настройки
- PD3 — прерывание INT1 — переключение между настраиваемыми параметрами

---

#### Как настроены прерывания в коде

c

```c
MCUCR = (1<<ISC11) | (1<<ISC10) | (1<<ISC01) | (1<<ISC00);
GICR  = (1<<INT1)  | (1<<INT0);
```

MCUCR — регистр управления режимом срабатывания. Биты ISC задают когда именно срабатывает прерывание:

```
ISC01=1, ISC00=1 → INT0 срабатывает по нарастающему фронту
ISC11=1, ISC10=1 → INT1 срабатывает по нарастающему фронту
```

Нарастающий фронт — это момент когда сигнал переходит из 0 в 1. То есть прерывание срабатывает в момент нажатия кнопки.

GICR — регистр разрешения прерываний. INT0 и INT1 = 1 — разрешаем оба прерывания.

---

#### ISR INT0 — переключение режимов

c

```c
ISR(INT0_vect) {
    if (mode == 0) {
        mode = 1;
        PORTA = 0x00;
        PORTB = 0x00;
        PORTC = 0x00;
        ind[0]=13; ind[1]=11; ind[2]=18; ind[3]=pb;
    }
    else {
        mode = 0;
        cur_b = bb;
        init_garlands();
    }
}
```

Разберём каждую ветку:

Если были в режиме гирлянды (mode == 0) — переходим в настройку:

- mode = 1 — переключаем режим
- Гасим все порты — останавливаем гирлянду
- Заполняем массив ind — показываем на индикаторе текущий параметр pb

Если были в режиме настройки (mode == 1) — возвращаемся в гирлянду:

- mode = 0 — переключаем режим
- cur_b = bb — сбрасываем текущую маску порта B к базовому значению
- init_garlands() — применяем начальные смещения и запускаем гирлянду

---

#### Что делает init_garlands

```c
void init_garlands() {
    crcl(&cur_c, hc);
    update_garland();
}
```

Применяет начальное смещение для порта C — сдвигает cur_c на hc бит. Потом выводит значения на порты.

По варианту 5 начальные смещения: da=0, db=0, dc=1. Поэтому только порт C сдвигается один раз при старте.

---

#### Про INT1 — переключение параметров

В коде ISR для INT1 не написан — это значит что в варианте 5 настраивается только один параметр (pb), и переключаться между параметрами не нужно. Кнопка PD3 в данном варианте не используется.
