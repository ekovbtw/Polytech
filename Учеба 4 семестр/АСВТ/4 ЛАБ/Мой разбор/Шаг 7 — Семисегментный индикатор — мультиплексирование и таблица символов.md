#### Что такое семисегментный индикатор

Это дисплей из 7 светодиодных сегментов, которые складываются в цифры и буквы. Каждый сегмент — это отдельный светодиод с именем от a до g.

---

#### Важный момент — общий анод

В коде индикатор с общим анодом. Это значит логика перевёрнута — 0 зажигает сегмент, 1 гасит. Поэтому в таблице символов все значения инвертированы:

c

```c
const unsigned char numb[] = {
    0b11000000, // 0 — горят все сегменты кроме g (среднего)
    0b11111001, // 1 — горят только b и c
    ...
    0b11111111  // ' ' — пробел, все сегменты погашены
};
```

---

#### Проблема — 4 индикатора, один порт

На плате 4 семисегментных индикатора. Если подключить каждый отдельно — нужно 4×7 = 28 ножек. Но у нас их нет столько. Решение — мультиплексирование.

Идея простая: все 4 индикатора подключены к одному порту C (сегменты). А порт A управляет питанием каждого индикатора по очереди. В любой момент горит только один индикатор, но они переключаются так быстро, что глаз видит все четыре одновременно.

#### Как это реализовано в коде

В прерывании таймера T0, когда mode == 1:

c

```c
PORTA = 0x00;                    // 1. гасим ВСЕ индикаторы
if (++cur_ind > 3) cur_ind = 0;  // 2. переходим к следующему (0→1→2→3→0)
PORTC = ~numb[ind[3 - cur_ind]]; // 3. выставляем символ
PORTA = (1 << cur_ind);          // 4. включаем нужный индикатор
```

Шаг 1 — гасим всё, чтобы при смене символа не было мусора на экране

Шаг 2 — переходим к следующему индикатору по кругу

Шаг 3 — берём нужный символ из массива ind, смотрим его код в таблице numb, инвертируем через тильду потому что индикатор с общим анодом

Шаг 4 — включаем питание нужного индикатора через PORTA

---

#### Почему ind[3 - cur_ind]

Индикаторы на плате идут справа налево, а массив ind хранит цифры слева направо. Поэтому нужно зеркалить индекс. При cur_ind = 0 берём ind[3] — это самый правый индикатор.

---

#### Что хранится в массиве ind

Это индексы символов из таблицы numb. Для режима настройки pb:

c

```c
ind[0] = 13;  // символ 'p'
ind[1] = 11;  // символ 'b.' (с точкой)
ind[2] = 18;  // символ ' ' (пробел)
ind[3] = pb;  // само значение, например 3
```

На экране отображается "pb. 3"
