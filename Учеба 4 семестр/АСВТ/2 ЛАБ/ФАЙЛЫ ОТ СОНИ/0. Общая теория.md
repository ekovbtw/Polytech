### порты ввода-вывода

Микроконтроллер ATmega32 общается с внешним миром через **порты A, B, C, D** — каждый по 8 бит (8 ножек).
Каждый порт управляется тремя регистрами:

| Регистр | Назначение                                       |
| ------- | ------------------------------------------------ |
| `DDRx`  | Направление: **0** = вход, **1** = выход         |
| `PORTx` | Выход: записать значение. Вход: включить pull-up |
| `PINx`  | Чтение текущего состояния ножки                  |
Пример:
```c
DDRA  = 0xFF;  // все 8 бит PORTA = выход
PORTA = 0xFF;  // подать 1 на все → все светодиоды горят

DDRC  = 0x00;  // все 8 бит PORTC = вход
uint8_t val = PINC; // читаем что на ножках
```

### Pull-up резистор

Когда ножка настроена на **вход** и ни к чему не подключена — она "висит в воздухе" и читает случайные значения. Чтобы этого не было, включают **pull-up** (подтяжку к питанию):
```c
DDRC  = 0x00;  // вход
PORTC = 0xFF;  // включить pull-up
```

С pull-up:
- кнопка **не нажата** → читается **1**
- кнопка **нажата** (замыкает на землю) → читается **0**

### Прерывания

Прерывание — это когда происходит какое-то событие (нажата кнопка, переполнился таймер) и МК **прерывает основную программу** и выполняет специальную функцию — **ISR (обработчик прерывания)**.

Чтобы прерывания работали нужно:

1. Разрешить конкретное прерывание (например `TIMSK |= (1<<OCIE0)`)
2. Разрешить прерывания глобально — `sei()`

В нашем коде используются три прерывания:

|Прерывание|Когда срабатывает|Что делает|
|---|---|---|
|`TIMER0_COMP`|каждые ~1.2 мс|мигает гирлянда|
|`INT0`|нажата PD2|режим вперёд|
|`INT1`|нажата PD3|режим назад|
