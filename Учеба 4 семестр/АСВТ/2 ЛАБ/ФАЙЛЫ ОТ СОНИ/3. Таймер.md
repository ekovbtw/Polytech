```c
#define TIMER_TICKS_HALF_PERIOD  200

void timer0_init(void)
{
    TCCR0 = (1<<WGM01) | (1<<CS01) | (1<<CS00);
    OCR0  = 154;
    TIMSK |= (1<<OCIE0);
}

ISR(TIMER0_COMP_vect)
{
    tick_count++;
    if (tick_count >= TIMER_TICKS_HALF_PERIOD) {
        tick_count = 0;
        state ^= 1;
        uint8_t val = get_value();
        PORTA = val;
        PORTB = val;
        if (state) PORTD |=  (1<<PD4);
        else       PORTD &= ~(1<<PD4);
    }
}
```

#### Зачем нужен таймер?
Нам нужно мигать гирляндой с частотой **2 Гц** — то есть менять состояние каждые **0.25 секунды**. Делать это через `_delay_ms(250)` нельзя — программа зависнет и не сможет обрабатывать кнопки. Поэтому используем **таймер с прерыванием**.

#### Режим CTC
Таймер T0 работает в режиме **CTC (Clear Timer on Compare)**:

```
Счётчик считает: 0 → 1 → 2 → ... → OCR0 → сброс → 0 → 1 → ...
                                        ↑
                              здесь срабатывает прерывание
```
Как только счётчик достигает значения `OCR0` — срабатывает прерывание `TIMER0_COMP` и счётчик сбрасывается в 0.

#### Расчёт частоты прерываний
```c
TCCR0 = (1<<WGM01) | (1<<CS01) | (1<<CS00);
//        CTC режим    прескалер /64
OCR0  = 154;
```

Формула:
```
F_прерываний = F_CPU / (прескалер × (OCR0 + 1))
             = 8000000 / (64 × 155)
             = 8000000 / 9920
             ≈ 806 Гц
```
Значит прерывание срабатывает **806 раз в секунду**, то есть каждые ~1.24 мс.

#### Как из 806 Гц получить 2 Гц?

Нам нужно менять состояние каждые **0.25 секунды**. За 0.25 секунды таймер срабатывает:
```
806 × 0.25 ≈ 200 раз
```

Поэтому в ISR считаем тики:
```c
tick_count++;
if (tick_count >= 200) {  // прошло 0.25 секунды
    tick_count = 0;
    state ^= 1;           // меняем состояние
    ...
}
```

#### что делает ISR

```c
ISR(TIMER0_COMP_vect)
{
    tick_count++;                          // считаем тики
    if (tick_count >= 200) {
        tick_count = 0;                    // сбрасываем счётчик
        state ^= 1;                        // 0→1 или 1→0
        uint8_t val = get_value();         // получаем значение для текущего режима и состояния
        PORTA = val;                       // выводим на гирлянду
        PORTB = val;
        if (state) PORTD |=  (1<<PD4);    // PD4 = состояние
        else       PORTD &= ~(1<<PD4);
    }
}
```

`^=` — это **XOR**:
```
state = 0: 0 XOR 1 = 1  → стало 1
state = 1: 1 XOR 1 = 0  → стало 0
```
Идеальный способ переключать между 0 и 1.

#### Почему `TIMSK |= (1<<OCIE0)`?

```c
TIMSK |= (1<<OCIE0);  // разрешить прерывание по совпадению таймера T0
```

`TIMSK` — регистр маски прерываний таймеров. Бит `OCIE0` разрешает прерывание `TIMER0_COMP`. Без этой строки таймер будет считать, но прерывание не будет вызываться.
