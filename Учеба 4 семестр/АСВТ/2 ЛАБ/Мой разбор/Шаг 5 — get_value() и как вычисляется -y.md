### Что делает эта функция

get_value() — это сердце гирлянды. Она смотрит на текущий режим и текущее состояние, и возвращает байт, который нужно выдать на PORTA и PORTB.

```
uint8_t get_value(void)
{
    switch (mode) {
        case 1:
            return (state == 0) ? 0xFF : 0x00;
        case 2:
            return (state == 0) ? 0xAA : 0x55;
        case 3:
        default:
            return (state == 0) ? y : (y ^ 0x80);
    }
}
```

Режимы 1 и 2 понятны — просто возвращают фиксированные числа. Интереснее режим 3.

---

### Режим 3: что такое -y в прямом коде

По заданию вариант 5 — прямой код (буква "а" в номере варианта). Это значит -y вычисляется по правилу прямого кода.

Из методички — прямой код числа:

- старший бит (бит 7) кодирует знак: 0 = плюс, 1 = минус
- остальные биты — само число, одинаковое для +y и -y

Пример: если y = 0x55 = 0b01010101, то -y в прямом коде = 0b11010101 = 0xD5. Мы просто инвертировали бит 7 (знаковый), остальное не тронули.
