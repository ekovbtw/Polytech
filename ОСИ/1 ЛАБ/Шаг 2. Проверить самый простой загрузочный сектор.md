Сначала собрать и запустить **минимальный пример**, который просто пишет текст на экран.
возьму его из методички:
`use16`
`org 0x7C00`

`start:`
    `; Инициализация сегментов`
    `; Это не всегда обязательно для любого BIOS,`
    `; но так делать рекомендуется`
    `mov ax, cs`
    `mov ds, ax`
    `mov ss, ax`
    `mov sp, start`

    ; BIOS video interrupt:
    ; AH = 0x0E -> вывести символ на экран
    mov ah, 0x0E

    mov al, 'H'
    int 0x10

    mov al, 'e'
    int 0x10

    mov al, 'l'
    int 0x10

    mov al, 'l'
    int 0x10

    mov al, 'o'
    int 0x10

`inf_loop:`
    `jmp inf_loop`

    ; Заполняем сектор нулями до 512 байт минус 2 байта сигнатуры
    times 512 - ($ - $$) - 2 db 0

    ; Сигнатура загрузочного сектора
    db 0x55, 0xAA

Попробую его закомпилить командами: 
`yasm -f bin -o bootsect.bin bootsect.asm`

`qemu-system-i386 -fda bootsect.bin`
