# 1. Сумма выборки
 Сумма элементов выборки — это сумма всех значений случайной величины в выборке.

![[Pasted image 20260319165521.png]]

`def all_summ(X):`  
	`summ = 0`  
	`for x in X:`  
	`summ += x`  
	`return summ`

тут все просто и понятно
# 2. Выборочное среднее
Выборочное среднее — это среднее арифметическое всех элементов выборки. Оно показывает "центр" данных.

![[Pasted image 20260319165338.png]]

`def viborochnoe_avarage(X):`  
	`return all_summ(X) / len(X)`

# 3. Медиана
Медиана — это значение, которое делит упорядоченную выборку пополам.

![[Pasted image 20260319165918.png]]

Для начала отсортируем выборку (сортировка пузырьком):
`def my_sort(X):`  
    `n = len(X)`  
    `X_sorted = X.copy()  # чтобы не портить исходный список`  
  
    `for i in range(n):`  
        `for j in range(0, n - i - 1):`  
            `if X_sorted[j] > X_sorted[j + 1]:`  
                `# меняем местами`  
                `temp = X_sorted[j]`  
                `X_sorted[j] = X_sorted[j + 1]`  
                `X_sorted[j + 1] = temp`  
  
    `return X_sorted`
Теперь напишем функцию для поиска медианы:
`def median(X):`  
	`X_sorted = my_sort(X)`  
	`n = len(X)`  
	  
	`if n % 2 == 1:`  
	`return X_sorted[n // 2]`  
	`else:`  
	`return (X_sorted[n // 2 - 1] + X_sorted[n // 2]) / 2`

# 4. Мода
Мода — это значение, которое встречается в выборке чаще всего.

`def mode(X):`  
	`counts = {}`  
	  
	`for x in X:`  
	`if x in counts:`  
	`counts[x] += 1`  
	`else:`  
	`counts[x] = 1`  
	  
	`max_count = 0`  
	`mode_value = None`  
	  
	`for key in counts:`  
	`if counts[key] > max_count:`  
	`max_count = counts[key]`  
	`mode_value = key`  
	  
	`return mode_value`

# 5. Размах выборки
Размах — это разница между максимальным и минимальным значениями

`def sample_range(X):`  
	`return max(X) - min(X)`
	
Показывает, насколько "растянута" выборка.

# 6. Смещённая дисперсия
Смещённая дисперсия — это средний квадрат отклонений от среднего.

![[Pasted image 20260319203345.png]]

`def biased_variance(X):`  
	`mean = viborochnoe_avarage(X)`  
	`s = 0`  
	  
	`for x in X:`  
	`s += (x - mean) ** 2`  
	  
	`return s / len(X)`
	
# 7. Несмещённая дисперсия
Несмещённая дисперсия — более точная оценка дисперсии, учитывающая малый размер выборки.

![[Pasted image 20260319203554.png]]

`def unbiased_variance(X):`  
	`mean = viborochnoe_avarage(X)`  
	`s = 0`  
	  
	`for x in X:`  
	`s += (x - mean) ** 2`  
	  
	`return s / (len(X) - 1)`

# 8. Начальный момент k-го порядка
Начальный момент показывает среднее значение степеней элементов выборки.

![[Pasted image 20260319203941.png]]

`def raw_moment(X, k):`  
	`s = 0`  
	  
	for x in X:  
	s += x ** k  
	  
	return s / len(X)

# 9. Центральный момент k-го порядка
Центральный момент измеряет отклонение значений от среднего в степени k

![[Pasted image 20260319204204.png]]

`def central_moment(X, k):`  
	`mean = viborochnoe_avarage(X)`  
	`s = 0`  
	  
	for x in X:  
	s += (x - mean) ** k  
	  
	return s / len(X)
